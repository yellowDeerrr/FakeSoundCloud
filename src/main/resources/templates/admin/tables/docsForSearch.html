Documentation: Search in Table

This page contains a brief documentation on how to use the search function in a table.

How to Use the Search

To search for specific records in the table, use the input field and the "Search" button.

Examples of Search Queries

You can use the following formats for search queries:

  • id: Search by record ID (e.g., "id: 123").
  • log time: Search by log timestamp (e.g., "log time: 2023-09-17 10:30:00").
  • log level: Search by log level (e.g., "log level: INFO").
  • message: Search by message content (e.g., "message: Error occurred").
  • user id: Search by user ID (e.g., "user id: 456").

Special Query

If you leave the search query field empty and click "Search," all records in the table will be displayed.

Examples of Usage

Below are examples of search queries:

  • Search by ID: "id: 123"
  • Search by log level: "log level: ERROR"
  • Search by message: "message: Database connection failed"
  • Search by user ID: "user id: 789"

Please use these examples to perform searches in the log table on the "Backend Logs, front-end logs etc." page.
